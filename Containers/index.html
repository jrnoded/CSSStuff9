Container Query Not Supported.

Studying the default code I found that everything is wrapped in a the "page-wrap" class which also has the container-type element on it. Having the container type at that level is just the same as having the container on the body. Makes the @container the same as the a @media query. This makes no sense as the container is the same size as the window. Changing the sidebar to a fixed 350px keeps it from shrinking.
